The JobPlot Interface

The JobPlot interface will have all of the same selection widgets as the JobList interface. It will also have widgets that will affect the plot's display. These settings will be saved and used the next time the interface is opened. The user can change the Graph type to one of sticks, lines, steps or none. The user can also turn one or both of the plots off with the File Data and the Byte Data check boxes. The symbol type for each plot's points can be changed as well.
